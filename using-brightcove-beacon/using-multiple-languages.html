앱의 언어 선택

사전 판매 과정에서 앱을 여러 언어로 번역하겠다는 의사를 밝혔습니다. 번역 가능성에는 앱 필드 및 콘텐츠 메타데이터가 포함됩니다.

앱 번역은 앱 경험에서 앱 필드 번역을 요청하여 수행됩니다. 콘텐츠 번역은 CMS에 입력하는 메타데이터입니다. 여기에는 제목 및 설명과 같은 텍스트 데이터와 텍스트가 포함된 이미지가 포함됩니다.

Brightcove Beacon 앱에서 사용하려는 언어를 정의하는 것은 온보딩 프로세스의 일부입니다. 온보딩 프로세스가 끝난 후 여러 언어를 사용하기로 선택하거나 번역이 있는 언어를 추가하려면 Brightcove 지원 담당자에게 문의하십시오. 새로운 언어 번역에 대해서는 Brightcove 지원에 문의하십시오.

무엇을 번역합니까?

번역 가능/번역할 수 있는 앱의 두 부분은 다음과 같습니다.

  • 앱의 필드 이름, 버튼 및 설명에 대한 번역은 귀하가 제공하거나 Brightcove에서 귀하를 대신하여 번역할 수 있습니다.
  • 예를 들어 페이지 이름 또는 비디오 메타데이터 및 이미지와 같이 정의한 항목의 번역은 귀하가 Brightcove Beacon에 입력합니다.

시청자가 앱에서 사용할 언어를 선택할 수 있는 세 가지 방법이 있습니다. 그들은:

앱에서 언어 변경

시청자는 Brightcove Beacon 앱 자체에서 언어를 선택할 수 있습니다. 다음은 뷰어가 다음을 사용하여 따를 프로세스를 보여줍니다. 설정앱 언어를 영어에서 독일어로 변경하는 페이지:

브라우저에서 기본 언어 설정

뷰어가 브라우저에서 기본 언어를 설정하고 해당 언어가 Brightcove Beacon 앱에서 지원되는 경우 앱은 자동으로 해당 언어를 사용합니다. 뷰어는 언어를 설정할 필요가 없습니다. 설정표준 페이지.

메모: 실제 Brightcove Beacon 앱에서 언어를 설정하면 브라우저 설정이 무시됩니다.

널리 사용되는 모든 브라우저에는 기본 언어를 설정하는 방법이 있습니다. 예를 들어 다음은 Chrome에서 설정이 수행되는 방식입니다. 고급의섹션설정옵션:

기기의 언어 설정

예를 들어 iPhone에서 다음으로 이동합니다. 설정 → 일반 → 언어 및 지역

메모: 실제 Brightcove Beacon 앱에서 언어를 설정하면 장치 설정이 무시됩니다.

시청자가 동영상 자산 언어를 선택하는 방법

앱에서 사용할 수 있는 언어 외에도 비디오 자산에 사용할 수 있는 다국어 캡션 및 오디오 트랙이 있을 수 있습니다. 비디오에 사용 가능한 언어가 여러 개 있는 경우 뷰어는 비디오 재생 중에 원하는 언어를 선택합니다. 재생 중에 톱니바퀴 아이콘을 사용할 수 있으며 이 아이콘을 클릭하면 대화 상자가 표시되고 뷰어는 다음과 같이 자막 및/또는 오디오 트랙의 언어를 선택할 수 있습니다.

비디오 정보 현지화

비디오 주변에 현지화할 수 있는 항목이 많이 있습니다. 각각에 대한 세부 정보는 다음과 같습니다.

비디오 메타데이터 언어 정의

비디오 메타데이터를 현지화하는 기본 프로세스는 다음과 같습니다.

  • 에서 원하는 언어를 구성하십시오. 비디오 클라우드 스튜디오의 관리자기준 치수.
  • 비디오 속성 페이지에서 현지화된 메타데이터를 입력할 언어를 선택하십시오.
  • 현지화된 메타데이터를 입력합니다.

다국어 메타데이터 사용에 대한 자세한 내용은다국어 메타데이터 작업문서.

다국어 메타데이터를 사용하기 위한 첫 번째 단계는 다음으로 이동하여 Video Cloud Studio에서 메타데이터를 원하는 언어를 선택하는 것입니다. 관리자 → 언어 설정 . 다음과 같이 먼저 언어를 추가한 다음 저장합니다.

  1. 언어 추가
  2. 언어 저장

동영상 메타데이터

이 작업은 Video Cloud Studio에서 수행해야 합니다. 가다미디어그런 다음 다국어 메타데이터를 추가하려는 비디오를 클릭합니다. 실제로 메타데이터를 입력하려면영상정보섹션에서 언어를 선택하고 마지막으로 메타데이터를 입력합니다.

  1. 언어 선택
  2. 메타데이터 입력

비디오 이미지

이 작업은 Video Cloud Studio에서 수행해야 합니다. 가다미디어그런 다음 현지화된 이미지를 추가하려는 비디오를 클릭하십시오. 실제로 현지화된 이미지를 제공하려면영상정보섹션에서 언어를 선택하고 마지막으로 이미지를 업로드합니다.

비디오 자막/캡션

이 작업은 Video Cloud Studio에서 수행해야 합니다. 가다미디어모듈을 클릭한 다음 다국어 자막/캡션을 추가하려는 비디오를 클릭합니다. 실제로 자막/캡션을 제공하려면텍스트 트랙섹션, 클릭편집하다그런 다음 양식을 작성하십시오.

자세한 내용은비디오에 캡션 추가문서.

비디오 오디오 트랙

이 작업은 Video Cloud Studio에서 수행해야 합니다. 가다미디어그런 다음 오디오 트랙을 추가하려는 비디오를 클릭하십시오. 실제로 오디오 트랙을 제공하려면오디오 트랙섹션, 클릭편집 → 오디오 트랙 추가그런 다음 양식을 작성하십시오.

자세한 내용은비디오에 오디오 트랙 추가문서.
