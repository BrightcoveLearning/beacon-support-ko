    AuthVoD 구현

    이 항목에서는 AuthVod 수익 창출 패키지를 만드는 방법을 배우게 됩니다.

    AuthVod 이해하기

    AuthVod (AuthVod On Demand) 는 시청자가 패키지에 포함된 비디오를 시청하기 위해 브라이트코브 비콘 앱에 사용자 이름과 암호를 입력하여 인증해야 하는 수익 창출 패키지입니다. 해당 패키지를 구성하는 방법에 따라 동영상에 광고가 표시되거나 표시되지 않을 수 있습니다.

    AuthVod에 대한 자세한 정보와 브라이트코브 비콘에서 사용할 수 있는 모든 수익 창출 패키지에 대한 자세한 내용은 수익 창출옵션 이해문서를 참조하십시오.

    AuthVod 패키지 구성 개요

    AuthVod 패키지를 구성하는 상위 단계 단계는 다음과 같습니다.

    1. 브라이트코브 비콘에서게시되지 않은비용 없는 패키지를 만듭니다.
    2. 광고가 필요한 경우 패키지를 만들 때 광고 공급자를 제공하십시오.
    3. 비디오 클라우드 스튜디오에서 비디오의 사용자 정의 필드에 데이터를 추가하여 AuthVod 패키지에 할당합니다.
    4. 브라이트코브 비콘에서 비디오를 인제스트하고 캐시를 지웁니다.

    브라이트코브 비콘에서 패키지 만들기

    다음은 AuthVod 패키지를 만들기 위해 브라이트코브 비콘에서 수행해야 할 세부 단계입니다.

    1. 브라이트코브 비콘 상단에 있는커머스탭을 클릭합니다.
    2. 왼쪽 탐색에서패키지를선택합니다.
    3. 새 패키지 추가버튼을 클릭합니다.
    4. 패키지탭에서 내부 이름을 입력하고상태를게시되지않음으로 유지해야 합니다.
    5. 광고를 사용하려면예라디오 버튼광고그런 다음 선택광고 제공자드롭 다운에서.
    6. 텍스트 데이터탭에서 뷰어가 볼 수 있는 이름과 패키지에 대한 헤드라인을 입력합니다.
    7. 원하는 경우 패키지 정보와 함께 표시할 이미지를 제공합니다.
    8. Streams탭에서 동시에 스트리밍할 수 있는 최대 장치 수 및 활성 연결을 가질 수 있는 최대 장치 수에 대한 값을 입력합니다.
    9. 패키지 가용성을 더 세밀하게 제어하려면예라디오 버튼고급 스트림?을 클릭 한 다음 더 미세한 가용성 옵션으로 양식을 작성하십시오.

    Studio에서 패키지에 비디오 추가

    패키지에 콘텐츠를 추가하려면 Video Cloud Studio를 사용하여 사용자 정의 필드에 값을 할당해야 합니다. AuthVod 패키지에 대해 값을 할당해야 하는 두 개의 사용자 정의 필드는 다음과 같습니다.

    • 비콘. 권리.패키지 이름
    • 비콘. 권리. 유형

    아래 단계에서는 이러한 사용자 정의 필드에 값을 할당하는 프로세스를 자세히 설명합니다.

    브라이트코브 비콘과 함께 Video Cloud의 사용자 정의 필드를 사용하는 방법에 대한 자세한 내용은비디오 클라우드 사용자 정의 필드 만들기문서를 참조하십시오.

    패키지에 포함하려는 각 비디오에 대해 다음 단계를 완료하십시오.

    1. 비디오 클라우드 스튜디오에 로그인합니다.
    2. 미디어 모듈에서 비디오 이름을 클릭하여 해당 비디오의 속성을 봅니다.
    3. 사용자 정의 필드 ' 편집버튼을 클릭합니다.
    4. 비콘에 대한 권리.< >카운터.packagenName에서 이전에 만든 AuthVod 패키지의 이름을 입력합니다.
    5. 비콘에 대한 권리.< >카운터 유형에서SVOD를입력합니다.

    다음 스크린샷은 사용자 정의 필드에 사용된 실제 값의 예를 보여 줍니다. 이 경우카운터는 0 (빨간색으로 강조 표시) 이고 패키지 이름은AuthVoD 테스트 (노란색으로 강조 표시) 입니다. AuthVod를 구현할 때비콘.권한을 기억하십시오.< >카운터. type은항상SVOD로설정됩니다.

    브라이트코브 비콘으로 비디오 인제스트

    Video Cloud Studio의 동영상 변경 사항은 브라이트코브 비콘으로 수집해야 합니다. 이 작업은 매 시간마다 자동으로 수행되지만 필요에 따라 수동으로 수집 할 수도 있습니다. 광범위한 단계는 아래에 자세히 설명되어 있습니다.

    비디오 수집에 대한 자세한 내용은강제 동기화문서를 참조하십시오.
    1. 브라이트코브 비콘의 상단 메뉴에서 도구 (렌치 아이콘) 탭을 클릭합니다.
    2. 왼쪽 탐색에서 [ 수집 ] 옵션이 선택되어 있는지 확인합니다 (기본 옵션).
    3. 브라이트코브 비디오 업데이트버튼을 클릭합니다.
    4. 수집이 완료되면 먼저 상단 메뉴의상거래탭을 클릭하여 AuthVoD 패키지에 비디오가 할당되었는지 확인할 수 있습니다.
    5. 왼쪽 탐색에서패키지옵션이 선택되어 있는지 확인합니다 (기본 옵션).
    6. AuthVod 패키지의 이름을 클릭합니다.
    7. 컨텐츠탭을 클릭하면 이 AuthVod 패키지에 지정된 자산을 볼 수 있습니다. 이 탭의 예는 다음과 같습니다.
    8. 캐시 지우기를 시작하려면 상단 메뉴에서 도구 (렌치 아이콘) 탭을 클릭하십시오.
    9. 왼쪽 탐색에서캐시옵션을 클릭합니다.
    10. 캐시 비우기단추를 누릅니다.

    이제 브라이트코브 비콘 앱에 대해 인증된 모든 시청자는 AuthVod 콘텐츠에 액세스할 수 있습니다. 시청자가 인증되지 않은 상태에서 동영상을 재생하려고 하면 다음과 같이 동영상의 왼쪽 하단에 로그인하라는 메시지가 표시됩니다.
