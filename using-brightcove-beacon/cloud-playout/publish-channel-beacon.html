이 주제에 설명된 일부 작업은 Beacon에서 수행되고 다른 작업은 Video Cloud에서 수행됩니다.

소개

Cloud Playout 채널을 만든 후 Beacon 앱의 일부로 만들기 위해 수행해야 하는 세 단계가 있습니다. 다음 섹션에서 이에 대해 자세히 설명합니다.

비콘 채널에 대한 정보 얻기

계속하기 전에 Beacon 채널 및 재생 목록을 설정하는 데 필요한 정보를 캡처해야 합니다.

  1. 채널 목록에서 채널을 클릭하여 채널 세부 정보 페이지로 이동합니다.
  2. 아직 하지 않았다면 다음을 클릭하십시오. 활성화채널을 활성화합니다. 보다채널 작업 사용자세한 사항은.
  3. 채널이 활성화되면 기본 탐색을 사용하여 Video Cloud로 이동합니다. 미디어기준 치수.
  4. Cloud Playout 채널과 이름이 같은 비디오를 찾아야 합니다. 이미지가 없는 라이브 비디오가 됩니다.
  5. 비디오 이름을 클릭하여 세부 정보를 엽니다.
  6. 둘 다 저장비디오 ID그리고채널 ID (뒤에 오는 긴 영숫자 문자열클라우드 플레이아웃:태그에서).

비콘 채널 생성

이 단계는 현재 다음에서 수행되어야 합니다. 봉홧불 .

Brightcove Beacon에서 채널을 만들려면 다음 단계를 따르세요.

기본적인 정보

  1. 비콘 클래식에 로그인합니다.
  2. 상단 탐색에서채널 .
  3. 클릭새 채널 추가단추.
  4. 에서기본적인 정보탭, 정의이름당신의 채널을 위해.
  5. Cloud Playout 채널 ID 입력:

    Cloud Playout 채널 ID를 찾으려면 다음을 참조하십시오. 비콘 채널에 대한 정보 얻기 .

  6. 에서콘텐츠 등급드롭다운 메뉴에서 채널이 스트리밍될 국가의 콘텐츠 등급 시스템에 따라 채널의 콘텐츠 등급을 선택합니다. 예시 이미지에서는 미국용입니다. TV-Y .
  7. 에서장르 세부 정보섹션에서장르드롭다운 메뉴를 클릭하여 채널의 장르를 선택합니다. (예시: 공포, 드라마 등..)
  8. 채널에 맞는 장르를 찾지 못한 경우장르 만들기단추.
    9. NS 하위 장르드롭다운 메뉴는 현재 작동하지 않습니다.

텍스트 데이터

NS 텍스트 데이터섹션에서 채널에 표시 가능한 텍스트를 추가할 수 있습니다.

  1. 선택텍스트 데이터탭.
  2. 에서제목필드에서 채널 제목을 정의합니다.
  3. 에서설명필드에서 채널에 대한 설명을 정의합니다.
  4. 에서개요필드에서 채널에 대한 개요를 정의합니다.
    5. 추가할 수 있습니다텍스트 데이터채널에서 사용할 모든 언어에 대한 섹션입니다.

스트림

스트림 섹션에서 채널에 대한 스트리밍 정보를 설정합니다.

  1. 스트림 섹션에서브라이트코브로공급자클릭새 스트림 추가 :
  2. Steam 정보 추가:
    • 스트림 비디오 ID:채널의 라이브 스트림에 대한 비디오 ID 입력
    • 게시할 기기:모든 장치 선택/선택 취소를 클릭한 다음 게시하지 않으려는 장치를 제거합니다.
    • 스트림 유형:선택하다살다
    • DRM 유형:사용 중인 DRM 유형을 선택하거나열려있는
    • 스트림 형식: HLS 선택

권리 및 일정

  1. 에서권리 및 일정섹션에서 다음 정보를 추가합니다.
    • 상태:게시됨 선택
    • 다운로드 허용:시청자가 콘텐츠를 다운로드하도록 허용할지 여부를 선택합니다(대부분의 경우 대답은 아니오입니다).
    • 권리 유형:무료 또는 귀하의 콘텐츠에 적합한 것을 선택하십시오.
    • 시작 시간:채널을 사용 가능하게 만드는 시간(즉시 사용 가능하게 하려면 과거 날짜를 선택하십시오)
    • 종료 시간:채널을 사용할 수 없게 되는 시간(무한한 실행을 원하면 먼 미래로 설정할 수 있음)
    • 최대 스트림: 아무것도 입력할 필요가 없습니다. 0으로 두십시오.
    • 게시할 기기:모든 장치 선택/선택 취소를 클릭한 다음 게시하지 않으려는 장치를 제거합니다.
    • 허용된 위치:시작하다세계또는 보기를 허용한 영역
    • 거부된 위치:공백으로 두거나 보기가 허용되지 않는 영역을 입력하십시오.

채널 만들기

재생 목록에 채널 추가

비콘 페이지가 사용하기 때문에재생 목록콘텐츠의 경우 Cloud Playout 채널을 재생 목록에 추가해야 합니다. 재생 목록에는 다른 비디오가 포함될 수 있지만 일반적으로 페이지 콘텐츠를 구성하는 채널만 포함합니다. 보다재생 목록 만들기세부 사항을 위해.
