Cloud Playout 채널의 내장 코드 생성

Cloud Playout 채널에 대한 내장 코드를 생성하려면 라이브 모듈을 사용합니다. 라이브 모듈 사용에 대한 자세한 내용은 다음을 참조하십시오. 라이브 모듈을 사용하여 라이브 이벤트 생성 및 관리 .

두 가지 유형의 내장 코드를 사용할 수 있습니다.

  • 기준 - iframe 임베드 코드. 대부분의 경우에 권장됩니다. 이 유형은 플레이어를 <iframe> 태그 안에 배치합니다.
  • 고급의 - 인페이지 임베드 코드. 플레이어가 iframe이 아닌 HTML 페이지에 직접 존재하도록 허용합니다. 이를 통해 개발자는 플레이어와 관련 속성 및 이벤트에 쉽게 액세스할 수 있는 이점을 얻을 수 있습니다.

게시된 플레이어의 URL을 가져올 수도 있습니다. 이것은 브라우저에서 이벤트를 볼 때 유용합니다.

라이브 이벤트에 대한 내장 코드를 생성하려면 다음 단계를 따르세요.

  1. 라이브 모듈을 엽니다.
  2. 채널이 스트리밍될 라이브 이벤트를 찾습니다. 일반적으로 이벤트는대기탭에 이름이 있습니다. 채널채널이 생성될 때 스트리밍할 특정 이벤트를 선택하지 않는 한.
  3. 이벤트 이름을 클릭하여 엽니다. 제어실페이지.
  4. 딸깍 하는 소리게시 및 포함 > 플레이어 구성왼쪽 탐색에서
    메모: 채널이 서버 측 광고(SSAI)를 활성화하도록 구성된 경우 Brightcove Player 버전 6.18.3 이상 ~해야하다사용된다. 버전 6.18.3 이상 플레이어만 드롭다운 목록에 나타납니다.
  5. 클릭플레이어 URL브라우저에서 이벤트를 미리 보거나 사용할 적절한 내장 코드를 복사합니다.

프로그램을 만든 경우광고 삽입옵션 설정에 , 게시 옵션에는광고 구성 선택사용할 광고 구성을 선택할 수 있는 드롭다운입니다.
