새 페이지 만들기

이 주제에서는 새 Brightcove Beacon 페이지를 만드는 방법을 배웁니다.

개요

Brightcove Beacon 페이지는 사용자가 비디오 콘텐츠를 선택하고 시청하는 곳입니다. 캐러셀은 Video Cloud Studio에서 정의한 비디오 또는 재생 목록 그룹을 표시합니다. Brightcove Beacon 관리 콘솔을 사용하여 콘텐츠 캐러셀이 있는 새 페이지를 만들 수 있습니다.

페이지 만들기

새 페이지를 만들려면 다음 단계를 따르세요.

  1. Brightcove Beacon에 로그인합니다.

  2. 상단 탐색에서페이지 .

  3. 선택새 페이지 추가버튼을 눌러 새 페이지를 만듭니다.

  4. 를 위해기본적인 정보탭에서 다음 필드 값을 입력합니다.

    • 시스템 제목 - 내부 시스템 페이지 이름. 이것은 사용자에게 표시되지 않지만 친숙한 URL을 생성하는 데 사용됩니다. 이후 단계에서 볼 수 있는 제목을 입력합니다.
    • 주문하다 - 홈페이지의 페이지 순서
    • 장르 유형 - 값:
      • 주인 - 메인 페이지
      • 아이 - 하위 페이지
    • 탐색 유형 - 페이지 템플릿 목록에서 선택
    • 프레젠테이션 유형 - 값:
      • 아이콘텍스트 - 내부에 텍스트가 있는 아이콘 표시
      • 상 - 아이콘 표시
      • 텍스트 - 텍스트 표시
    • 화면 레이아웃 - 값: (모든 탐색 유형에 화면 레이아웃이 있는 것은 아님)
      • 제목이 지정된 화면 레이아웃 - 제목이 있는 레이아웃
      • 제목 없는 화면 레이아웃 - 제목이 없는 레이아웃
    • 집이야? - 홈페이지일 경우 선택
  5. 다음으로 페이지에 블록을 추가합니다. 블록은 재생 목록의 컨테이너입니다. 로그인 및 로그아웃 유형과 같은 일부 탐색 유형에는 블록이 없습니다. 우리는 선택했습니다탐색 유형의 값으로화면 .

  6. 선택블록 추가단추. 버튼을 선택하면 오른쪽에 블록 종류가 나타납니다.

  7. 원하는 블록 유형을 선택합니다. 이 경우 우리는 다음을 선택합니다. 목록 차단 . 선택블록 추가원하는 블록과 연결된 버튼.

  8. 선택레이아웃 추가단추. 버튼을 선택하면 오른쪽에 레이아웃 종류가 나타납니다.

  9. 추천, 회전식 또는 격자형 등 원하는 레이아웃 유형을 선택합니다. 이 예에서는회전 목마 풍경 레이블형세.

  10. 선택재생 목록 추가단추. 버튼을 선택하면 오른쪽에 재생 목록 유형이 나타납니다.

  11. 재생 목록을 선택하십시오. 이 예에서는재생 목록 추가관련된 버튼기분재생 목록.

    Video Cloud Studio를 사용하여 재생 목록을 만들고 Video Cloud 계정에 동기화하면 여기에 표시됩니다.

  12. 우리는 그것을 볼 수 있습니다기분재생 목록이 캐러셀에 추가되었습니다. 드래그 앤 드롭하여 항목을 재정렬할 수 있습니다.

  13. 를 위해텍스트 데이터 , 비텍스트 데이터 , 태그그리고권리 및 일정탭, 참조공통 구성 설정문서.

  14. 선택새 페이지 만들기버튼을 눌러 새 페이지를 만듭니다. 당신 준비 다 됐어요. Brightcove Beacon 페이지에 새 캐러셀이 표시되어야 합니다.
