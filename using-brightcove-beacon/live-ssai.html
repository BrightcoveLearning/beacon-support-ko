    라이브 SSAI의 구현

    이 항목에서는 브라이트코브 비콘에서 라이브 서버 측 광고 구성을 관리하고 만드는 방법을 배웁니다.

    개요

    브라이트코브 비콘은 라이브 이벤트에서 서버 측 광고 삽입 (SSAI를) 지원하므로 라이브 스트림 중에 서버 측 광고를 요청하고 표시할 수 있습니다.

    프로세스의 상위 수준 뷰는 다음과 같습니다.

    1. Studio에서 광고 구성을 만듭니다.
    2. 라이브 이벤트에 대한 채우기 슬레이트를 만듭니다.
    3. SSAI가 활성화된 라이브 이벤트를 만듭니다.
    4. 브라이트코브 비콘에 라이브 이벤트를 게시합니다.
    이 문서는 라이브 이벤트에 SSAI를 추가하는 방법을 안내하는라이브 이벤트 스트리밍설명서를 보완합니다. 이벤트 생성 방법에 대한 자세한 내용은브라이트코브 비콘에서 라이브 이벤트 만들기섹션을 참조하십시오.

    SSAI는 라이브 이벤트 중에 다음과 같은 주요 기능을 제공합니다.

    • 광고는 서버측 라이브 동영상 스트림에 삽입되므로 광고 차단기의 영향을 받지 않습니다.
    • 광고는 인코더에서 전송된 큐 포인트에 삽입되거나 라이브 모듈 제어실을 사용하여 인스턴트 큐 포인트를 만들 수 있습니다.
    • 사용하지 않는 광고 시간을 채우도록채우기 슬레이트를구성할 수 있습니다.

    서버 측 광고 설정 구성

    여기에서는 브라이트코브 비콘 라이브 이벤트에 표시될 광고 세트로 사용할 SSAI광고 구성을 만듭니다.

    1. 스튜디오에 로그인합니다.
    2. 탐색 헤더에서ADMIN을클릭하고서버측 광고 설정을선택합니다. 모든 광고 구성 목록이 표시됩니다.
    3. 새 광고 구성을 만들려면 광고 구성만들기버튼을 클릭합니다.
      서버 측 라이브 광고 구성은 라이브 API를 사용하여 관리할 수도 있습니다. 자세한 내용은브라이트코브 라이브 API: 서버 측 광고 삽입 (SSAI를) .
    4. 드롭다운에서 유형으로Live를선택합니다.
    5. 서버측 광고 설정양식이 표시되는지 확인합니다.
    6. 광고 구성에이름을 지정합니다 .
    7. 광고 서버 응답을선택합니다.
      • 광대
      • DFP (광고 규칙)
      • 스마트 XML
      올바른 값을 선택할 수 있는지 광고 운영 관리자에게 확인하십시오.
    8. 광고 태그 (서버 URL) 를입력합니다.
    9. 저장버튼을 클릭합니다.

    성토 슬레이트 구성

    채우기 슬레이트는 라이브 스트림 광고 휴식 시간에 동적으로 게재되는 광고로 채울 수 없는 간격이 있을 때 표시되는 동영상입니다. 요청된 광고가 광고 휴식 시간보다 짧으면 시청자가 빈 플레이어를 보지 않도록 채우기 슬레이트가 표시됩니다.

    서버 측 광고를 요청하기 전에 채우기 슬레이트를 하나 이상 구성해야 합니다.

    채우기 슬레이트를 추가하려면 다음 단계를 수행합니다.

    1. Studio 헤더에서ADMIN > 라이브 설정링크를 클릭합니다.
      ADMIN메뉴에 액세스하려면 계정 관리자여야 합니다.
    2. + 채우기 슬레이트 추가를클릭합니다. 다음과 같은 양식이 표시되어야합니다.
    3. 채우기 슬레이트의이름을입력합니다.
    4. 슬레이트의소스 URL을입력합니다.
    5. 저장을 클릭합니다. 슬레이트의 섭취 과정이 시작됩니다.

    채우기 슬레이트를 삭제하려면 채우기 슬레이트 옆에 있는 삭제 버튼 ( ) 을 클릭합니다.

    SSAI를 지원하는 라이브 이벤트 만들기

    라이브 이벤트가 SSAI (서버측 광고 삽입) 를사용하려면 이벤트가 생성될 때 SSAI (서버 측 광고 삽입)설정을 선택하고 이전에 만든 채우기 슬레이트를 재생하도록 선택해야 합니다.

    브라이트코브 비콘에서 SSAI광고를 지원하는 이벤트를 생성하려면 다음 단계를 따르세요.

    1. Studio에서라이브모듈을 엽니다.
    2. 이벤트 관리 페이지에서 이벤트생성을누릅니다.
    3. 이벤트 세부 정보를 입력합니다. 라이브 모듈을 사용하여 라이브 이벤트를 만드는 방법에 대한 자세한 내용은 라이브 모듈을사용하여 라이브 이벤트 생성 및 관리를참조하십시오.
    4. 고급 옵션을클릭하여 고급 옵션을 확장합니다.
    5. 서버 측 광고 삽입 (SSAI를) 활성화를선택합니다. 이렇게 하면 서버 측 광고 삽입이 가능합니다.
    6. 성토 슬레이트를선택합니다. 실시간 스트림 광고 휴식 시간에 동적으로 게재되는 광고로 채울 수 없는 간격이 있는 경우 채우기 슬레이트가 표시됩니다.
    7. 광고 오디오설정을 선택하면 설정된 조정에 따라 광고의 오디오 레벨이 정규화됩니다. 이는 광고가 콘텐츠보다 크게 크지 않게 하는 데 도움이 되는 방법으로 사용될 수 있습니다. 브라이트코브는 이 옵션을 선택하는 것이 좋습니다.
    8. 이벤트 만들기를클릭합니다.
    9. 스트리밍 끝점 (RTMP URL)및스트림 이름등 라이브 이벤트에 대한 세부 정보가 포함된제어실페이지가 열립니다. 이 값은 인코더에서 사용됩니다.
    10. 인코더를 구성하고 라이브 스트림을 시작합니다.
      라이브 플랫폼에서 스트리밍을 시작하려면 인코더가 필요합니다. 인코더 사용 방법에 대한 자세한 내용은 다음 설명서를 참조하십시오.

      단계별: 텔레스트림 와이어캐스트와 함께 라이브 모듈 사용

      단계별: 오픈 브로드캐스터 소프트웨어 (OBS) 와 함께 라이브 모듈 사용

    11. 기본적으로제어실미리보기 플레이어에는 광고 없는 스트림이 표시됩니다. 라이브 광고 구성을 선택하지 않으면제어실에광고가 표시되지 않습니다. 아래 드롭다운을 사용하여 플레이어에게 광고 구성을 선택합니다.
    12. (선택 사항) 더하기 버튼 ( ) 을 클릭하여 추가 광고 구성을 선택합니다. 이렇게 하면제어실페이지에 다른 프리뷰 플레이어가 추가됩니다.

      프리뷰 플레이어 아래의 볼륨 버튼을 클릭하여 활성화된 플레이어 오디오 스트림을 제어합니다.

    광고 요청하기

    라이브제어실에서광고를 요청하려면 다음 단계를 따르세요.

    1. 고토 광고를클릭하고 드롭 다운에서 원하는 광고 나누기의 길이를 선택합니다.
    2. 삽입을클릭하여 실시간 스트림에 광고 나누기를 삽입합니다. 광고가 전체 광고 휴식 시간을 채우지 않으면 채우기 슬레이트 (이벤트가 생성될 때 선택됨) 가 표시됩니다.

    광고는제어실미리보기 플레이어와 라이브 모듈을 사용하여 게시한 플레이어에서 볼 수 있습니다. 이 예에서는제어실에표시되는 2개의 광고가 표시됩니다.

    브라이트코브 비콘에서 라이브 이벤트 게시

    브라이트코브 비콘에 서버 측 광고로 라이브 이벤트를 게시하려면 다음 단계를 따르십시오.

    1. 라이브컨트롤 룸의왼쪽 탐색에서게시 및 포함을클릭합니다.
    2. 플레이어 구성을선택합니다.
    3. 라이브 플레이어 옵션에서사용 가능한 플레이어를선택합니다.
    4. 서버 측 광고 설정에서 이전에 만든 광고구성을선택합니다.
    5. 사용 가능한 플레이어및광고 구성을선택하면 새 플레이어 URL이 생성됩니다. 라이브단어부터 시작하여 이미지에 표시된 것처럼 URL의 선택된 부분을 복사합니다. 예: 라이브.xxxxxxx
    6. 브라이트코브 비콘 플랫폼으로 이동합니다.
    7. 브라이트코브 비콘의 상단 메뉴에서광고탭을 클릭합니다.
    8. [ 새 구성 추가 ] 단추를 클릭합니다.
    9. 새 광고양식이 표시되는지 확인합니다.
    10. 새 광고 구성에이름을 지정합니다 .
    11. 라이브 광고 구성 ID:에 라이브 플랫폼에서 복사한플레이어 URL ID (예: live.xxxxx ) 를 붙여넣습니다.
    12. 제공업체메뉴에서브라이트코브 SSAI를선택합니다.
    13. 새 광고 만들기버튼을 클릭합니다.
    14. 상단 메뉴이벤트탭을 클릭하고 라이브 이벤트를 만들거나 선택합니다. 브라이트코브 비콘 라이브 이벤트를 만드는 방법에 대한 자세한 내용은라이브 이벤트 스트리밍문서를 참조하십시오.
      라이브 이벤트가 브라이트코브 비콘에서 성공적으로 스트리밍되려면작업 ID및스트림 비디오 ID가현재 라이브 이벤트의 ID와 일치해야 합니다.
    15. 브라이트코브 비콘 라이브 이벤트에서권한 및 일정탭을 클릭합니다.
    16. 권한 유형드롭다운 메뉴에서광고 - AVOD를선택합니다.
    17. 광고 구성하단 메뉴에서 이전에 생성한 광고 구성을 선택합니다.
    18. 이벤트 업데이트버튼을 클릭합니다.
    19. 화면 오른쪽 상단의도구 (렌치) 버튼을 클릭합니다. 그런 다음 캐시를 지웁니다.
    20. 브라이트코브 비콘 앱을 열고 이벤트를 선택합니다 (이미지의 이벤트 이름과 사진은 예시용으로만 사용).
    21. 브라이트코브 비콘 앱에서재생버튼을 클릭합니다. 이제 Video Cloud에서 SSAI가 삽입한 광고를 통해 라이브 이벤트 스트리밍을 볼 수 있습니다.

    이벤트가 종료되면 전체 또는 일부를 Video Cloud의 비디오 에셋이 되는 클립 또는 브라이트코브 비콘에 표시할 수 있는 재생 가능한 스트림으로 저장할 수 있습니다. 라이브 이벤트 광고는 저장되지 않지만, 녹화된 라이브 이벤트의 이후 VOD에 새 광고를 추가할 수 있도록 광고가 전송된 큐 포인트가 등록됩니다.

    클립을 만드는 방법에 대한 자세한 내용은라이브 이벤트에서 클립 만들기섹션에서라이브 이벤트 스트리밍문서.
