도구 사용

이 항목에서는 옵션의 기능을 배웁니다. 도구 Brightcove Beacon 섹션.

도구 소개

브라이트코브 비콘에서렌치아이콘(1으로 표시됨)은 도구 세트를 나타냅니다. 도구에 대한 옵션은음식물 섭취그리고은닉처 (2). 이 문서에서는 두 가지 옵션을 살펴봅니다.

음식물 섭취

수집 그룹에는 세 가지 도구가 있습니다.

  • Brightcove 비디오 업데이트 : Video Cloud 카탈로그에 비디오를 추가했거나 기존 비디오를 변경한 경우 이 버튼은 새로 추가되거나 업데이트된 비디오를 Brightcove Beacon에 동기화합니다.
  • Brightcove 재생 목록 업데이트 : Video Cloud 카탈로그에 재생 목록을 추가했거나 기존 재생 목록을 변경한 경우 이 버튼은 새로 추가되거나 업데이트된 재생 목록을 Brightcove Beacon에 동기화합니다.
  • XML EPG - 폴더에서 XML 가져오기 : EPG(Electronic Program Guide)를 업데이트해야 하는 경우 이 버튼은 업데이트된 XML 파일을 검색합니다. 파일을 삭제할 S3 계정이 제공됩니다.
또한 Brightcove 지원 담당자와 협력하여 수집을 자동으로 수행하는 일정을 만들 수도 있습니다.

섭취 시 주의 사항

Studio의 특정 비디오에 대해 입력된 모든 데이터가 자동으로 Brightcove Beacon에 수집되는 것은 아닙니다. 예를 들어 다음은 Video Cloud에서 Brightcove Beacon으로 수집되지 않습니다.

  • 이용 가능 시간 : 에 설정된 일정 시간 유효성 비디오 속성의 섹션은 Brightcove Beacon으로 수집되지 않습니다. 날짜는 되지만 시간은 아닙니다.
  • 지리적 필터링 : 에 설정된 지오 필터링 정보 유효성 비디오 속성의 섹션은 Brightcove Beacon으로 수집되지 않습니다.
위에 나열된 문제는 다음을 생성하여 해결할 수 있습니다. 패키지 Brightcove Beacon에서 사용자 정의 필드를 통해 수집 시 해당 패키지에 비디오를 할당합니다. 참조구독 패키지 만들기그리고구독 패키지 사용자세한 내용은 문서를 참조하십시오.

또한 Brightcove Beacon에 언급된 모든 필드에 연결된 사용자 정의 필드가 있는 것은 아닙니다. 예를 들어 생산 국가 그리고 하위 장르 연결된 해당 사용자 정의 필드가 없습니다.

은닉처

캐시 그룹에는 두 가지 도구가 있습니다.

  • 캐시 제거 : Brightcove Beacon에서 변경할 때 이 버튼을 클릭하여 해당 변경 사항을 현재 구성으로 롤링합니다. 이 옵션은 이미지를 제외한 모든 변경 사항을 현재 구성으로 가져옵니다.
  • 이미지 캐시 : Brightcove Beacon에서 이미지를 업데이트하는 경우 이 버튼을 클릭하여 이미지 변경 사항을 현재 구성으로 롤링합니다.

프로덕션 환경에서 이 작업을 너무 자주 수행하면 확장성과 성능에 부정적인 영향을 미치므로 이 프로세스를 무분별하게 수행해서는 안 됩니다.
