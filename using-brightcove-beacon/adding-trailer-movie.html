예고편 추가

Beacon 영화의 예고편을 추가하는 것은 간단합니다. 아래 단계를 따르세요.

  1. 예고편 비디오를 Video Cloud 계정에 업로드하십시오.
  2. 당신은 ~ 아니다예고편 비디오에 특별한 Beacon 사용자 정의 필드를 추가해야 합니다. 필요한 것은비디오 ID :
  3. Beacon 영화에 사용된 영상의 경우, 다음 사용자 정의 필드 추가 :
    예고편 사용자 정의 필드
    이름 표시하기 내부 이름 값
    beacon.trailer.id beacon_trailer_id (예고편의 비디오 ID)
