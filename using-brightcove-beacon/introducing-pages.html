페이지 내용

    소개 페이지

    이 항목에서는 브라이트코브 비콘 앱에서Pages사용에 대한 개요를 제공합니다. 페이지에는 앱 사용자가 볼 수 있는 기능과 비디오 콘텐츠가 포함되어 있습니다.

    페이지 유형

    페이지를 만들거나 편집할 때페이지 유형선택에서 다음을 제공합니다.

    페이지 카테고리

    페이지 유형은 세 가지 범주로 그룹화할 수 있습니다. 각 카테고리와 세부 사항은 다음과 같습니다.

    1. 구성 불가능
      • 앱에 포함시키거나 포함하지 않도록 선택한 페이지입니다.
      • 구성할 수 없는 페이지를 포함하는문서는 이러한 유형의 페이지 중 하나를 앱에 추가하는 방법을 포함하여 자세한 정보를 제공합니다.
      • 구성할 수 없는 페이지의 예로는 다음과 같이설정및가입이있습니다.
    2. 스크린
      • 동영상 콘텐츠, 회전 목마, 목록 등을 보관하는 페이지
      • 블록 , 레이아웃및재생 목록을사용하여 고도로 구성 할 수 있습니다. 여기에 두 가지 예가 나와 있습니다.
      • 스크린 영웅 이미지
        • 화면 페이지의 특별한 유형이지만 비디오 콘텐츠 뒤에 이미지를 허용합니다.
        • 다양한 플랫폼 (예: 웹 (가로) 및 모바일 (세로) 에 대해 가로 방향 및 포스터 방향 이미지를 모두 업로드해야 합니다.
        • 화면 상단의“영웅”영역은커버되지 않으며 구성 할 수 없습니다.
    3. 리디렉션
      • 리디렉션 페이지 유형은 웹 앱에만 해당됩니다.
      • 리디렉션 - 빈새 탭에서 웹 페이지를 엽니 다.
      • redirect-self는동일한 탭에서 웹 페이지를 엽니 다.
      • 외부 리디렉션 탭에서 URL을 지정합니다.

    직접 리디렉션 웹 전용; 메뉴 옵션을 클릭 한 리디렉션이 동일한 탭으로 발생하는 경우. 아니요 선택 사항
    화면- 그룹화된 비디오를 회전 목마 또는 탭으로 표시합니다. 앱의 콘텐츠를 구성하는 데 사용됩니다. 예 필수
    스크린 영웅 이미지 당신이 비디오 콘텐츠 뒤에 이미지를 추가 할 수있는 화면 페이지의 특별한 유형입니다. 예 선택 사항
    검색- 내용을 검색 한 다음 결과를 표시 할 수있는 입력 필드를 입력 화면을 제공합니다. 아니요 적극 권장

    화면 페이지 구성

    이 문서의 앞부분에서 언급했듯이화면페이지에는 비디오가 포함되어 있으며 비디오를 나타내는 이미지 표시 옵션을 제공합니다. 이 섹션에서는 해당 옵션이 앱에서 어떻게 보이는지에 대한 스크린샷과 함께 구성 옵션을 볼 수 있습니다. 먼저 화면 페이지에서 컨테이너의 최고 수준 인 블록에 대한 정보를 볼 수 있습니다. 그런 다음 비디오를 나타내는 이미지가 표시되는 방법을 제어하는 레이아웃 정보를 볼 수 있습니다.

    블록

    노란색으로 강조 표시된 블록 유형은 다음과 같습니다.

    다음은 각 유형의 이미지입니다.

    단일 블록

    단일 블록은하나의 레이아웃/재생 목록 조합을 블록에 추가 할 수 있습니다. 스크린 샷은 회전 목마를 표시하는단일 블록을보여줍니다. 단일 블록에여러 레이아웃/재생 목록 조합을 추가하거나 여러 레이아웃/재생 목록 조합이 포함 된단일 블록으로변경하려고하면 오류가 발생합니다.

    목록 블록

    목록 블록을사용하면 여러 레이아웃/재생 목록 조합을 블록에 추가 할 수 있습니다. 스크린 샷은 거기에 포함 된 세 개의 회전목마 목록 블록을보여줍니다.

    탭_블록

    탭 블록을사용하면 여러 레이아웃/재생 목록 조합을 블록에 추가 할 수 있지만 한 번에 하나만 표시되고 탭은 사용자가 볼 수있는 것을 선택할 수 있습니다. 다음 두 가지 제한 사항이 적용됩니다.

    • 테이블 블록 내부의 레이아웃은 동일해야합니다.
    • 테이블 블록 내부에서 배치는 그리드여야 합니다.
    스크린 샷은 재생 목록을 선택할 수있는테이블 블록을보여줍니다.

    드롭다운 블록

    드롭 다운 블록을 사용하면 블록에여러 레이아웃/재생 목록 조합을 추가 할 수 있지만 한 번에 하나만 표시되고 드롭 다운에서는 사용자가 볼 수있는 레이아웃을 선택할 수 있습니다. 다음 두 가지 제한 사항이 적용됩니다.

    • 드롭 다운 블록 내부의 레이아웃은 동일해야합니다.
    • 드롭 다운 블록 내에서 레이아웃은 그리드이어야합니다.
    스크린 샷은 세 개의 재생 목록을 선택할 수있는드롭 다운 블록 (노란색 사각형 내부) 을 보여줍니다.

    레이아웃

    선택한 레이아웃 위의 모든 블록 예제에서 회전 목마가있었습니다. 회전 목마는 가로 목록에 각 비디오의 이미지를 표시합니다. 이것이 당신이 선택할 수있는 유일한 레이아웃이 아닙니다. 사용 가능한 레이아웃은 노란색으로 강조 표시됩니다.

    각 레이아웃 유형의 스크린샷이 아니라 각 레이아웃 특성의 예가 표시됩니다. 위의 노란색으로 강조 표시된 유형은 서로 다른 특성의 조합입니다.

    회전 목마 및 레이블

    회전 목마는 이미지를 가로 행으로 표시하고 레이블은 이미지의 작은 아래쪽 부분을 오버레이합니다.

    라벨 출력

    라벨은 이미지 아래에 있습니다.

    주요기능 소개

    이미지는 더 큽니다 (빨간색 박스형, 그 아래 일반 회전 목마). 항상 그렇지는 않지만 추천 레이아웃은 페이지 상단에 사용됩니다.

    가로

    이미지는 가로 비율에 맞게 조정됩니다.

    포스터

    이미지는 포스터 (세로) 비율이 조정됩니다.

    그리드 3 및 그리드 4

    두 가지 유형의 그리드는 2차원 행/열 패턴으로 이미지를 표시합니다. 그리드 뒤에 오는 숫자 (3 또는 4) 는 이미지의 상대적 크기를 나타냅니다. 항상 표시 될 열 수를 나타내지 않습니다. 이 스크린 샷은 800px 너비의 브라우저를 보여 주며 그리드 3에는 세 개의 이미지의 두 행이 있습니다. 그리드 4는 이미지가 작기 때문에 가로 질러 네 개의 이미지를 표시하고있다.

    이 다음 스크린 샷은 여전히 그리드 3과 그리드 4를 모두 사용하고 있지만 브라우저 너비는 1600px입니다. 그리드 3 이미지는 이제 네 개의 가로 세로되어 있으며 Grid 4 레이아웃의 5개의 이미지가 화면에 맞을 수 있습니다. 다시 그리드 3 이미지가 더 큰 것을 볼 수 있습니다.

    이렇게 하면 블록 및 배치에 대한 옵션이 완료됩니다.

    일반 페이지 구성 설정

    브라이트코브 비콘에서 새 페이지를 만들거나 기존 페이지를 편집할 때 다음 탭이 표시됩니다.

    기본 데이터탭에서는 페이지 유형을 선택하고화면페이지를 구성할 수 있습니다. 다른 탭은 모든 페이지 유형에 대해 동일합니다. 그만큼공통 구성 설정문서에 필요한 정보를 자세히 설명합니다. 텍스트 데이터 , 비 텍스트 데이터 , 태그과권리 및 일정탭.
