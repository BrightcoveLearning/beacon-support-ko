    가입 패키지 작성

    이 항목에서는 브라이트코브 비콘의Commerce탭을 사용하여 비디오 구독 패키지를 구성하는 방법에 대해 설명합니다. 서브스크립션 패키지는 SVOD (온디맨드 서브스크립션 비디오) 수익 창출을 구현합니다. 구독 패키지를 비디오에 적용하는 방법을 알아보려면구독 패키지 사용문서를 참조하십시오.

    개요

    구독 패키지는 앱 고객이 특정 동영상 세트를 구매할 수 있는 방법을 정의합니다. 패키지를 만들 때 예를 들어 다음을 구성할 수 있습니다.

    • 청구 빈도
    • 광고가 지원되는지, 사용할 광고 구성 설정
    • 결제 방법
    • 패키지의 영향을 받는 비디오를 동시에 시청할 수 있는 최대 스트림 수
    • 패키지를 구입할 수 있는 시작 및 종료 날짜
    • 허용된 장치 및 지리적 위치

    패키지를 만드는 단계

    새 패키지를 만들려면 다음 단계를 수행하십시오.

    1. 브라이트코브 비콘에서상거래탭을 선택하고 왼쪽 탐색에서패키지가선택되어 있는지 확인한 다음새 패키지 추가를클릭합니다.
    2. 패키지탭 (아래 스크린샷) 에서 다음을 완료합니다.
      • 이름 : 비디오의 Video Cloud 사용자 정의 필드에 사용되는 패키지의 이름입니다.
      • 청구 빈도 : 이 패키지의 청구 빈도를 선택합니다.
      • 평가판 기간 : 이 패키지의 평가판 기간입니다. 평가판 기간이 없는 경우0을입력합니다.
      • 상태 : 패키지를 게시하여 사용할 수 있도록 하려면 선택합니다. 하나 이상의 결제 방법을 선택해야만 패키지를 게시할 수 있습니다.
      • 광고 : 이 패키지에서 광고를 사용할지 선택합니다.
      • 광고 제공업체 : 이전 단계에서 광고를 사용하도록 선택한 경우 광고 공급자를 선택합니다. 이 드롭다운은 브라이트코브 비콘의광고섹션에 나열된 광고 구성으로 채워집니다.
      • 사용할 결제 방법 : 이 패키지에 사용할 결제 방법을 선택합니다. 선택한 각 방법에 대해 결제 방법 목록 아래에 섹션이 열리므로 선택한 각 방법에 대해 완료해야 합니다. 아래 스크린샷은스트라이프 (노란색)와Apple Store (빨간색)가 선택되어 있음을 보여줍니다.
    3. [ 패키지 생성 ] 단추를 클릭하여 현재 패키지 구성을 저장합니다.
    4. 텍스트 데이터탭에서 다음을 완료합니다.
      • 팩 이름 : 사용자에게 표시되는 이름입니다.
      • 헤드라인 : 헤드라인은 사용자에게 표시됩니다 (예: 연말연시 특별 광고)! .
    5. [ 패키지 생성 ] 단추를 클릭하여 현재 패키지 구성을 저장합니다.
    6. 비텍스트 데이터탭에서 패키지 이름 옆에 사용할 이미지를 제공합니다.
    7. [ 패키지 생성 ] 단추를 클릭하여 현재 패키지 구성을 저장합니다.
    8. 컨텐츠탭은 현재 시간에 사용할 수 없습니다.
    9. Streams탭에서 다음을 완료합니다.
      • 최대 스트림 : 동시에 스트리밍할 수 있는 연결된 최대 장치 수입니다.
      • 최대 디바이스 수 : 활성 연결을 가질 수 있는 연결된 최대 장치 수입니다.
        위의 두 설정도 전역으로 설정되므로 이 패키지의 전역 설정을 재정의하려면 이 설정을 사용하십시오.
      • 고급 스트림 : 이 패키지가 설정하는 권한을 사용자 정의하려면 이 옵션을Yes로설정합니다. 사용 가능한 변경 사항은 다음 스크린 샷에 나와 있습니다:
    10. 새로 구성된 사용자 지정 권한을 저장하려면새 오른쪽 추가단추를 클릭합니다.
    11. [ 패키지 생성 ] 단추를 클릭하여 현재 패키지 구성을 저장합니다.

    그것이 패키지 생성을위한 것입니다!

    스트라이프 쿠폰 생성

    패키지를 구매할 때 적용할 수 있는 할인 (고정 금전적 할인 또는 백분율 할인) 을 제공하기 위해 쿠폰을 만들 수 있습니다.

    1. 브라이트코브 비콘에서커머스탭을 선택한 다음스트라이프로 쿠폰을클릭하고 마지막으로새 바우처 추가를클릭합니다.
    2. 새 쿠폰 생성양식 (아래 스크린샷) 에서 다음을 완료합니다.
      • ID : 시청자가 쿠폰을 사용하기 위해 브라이트코브 비콘 앱에 입력하는 코드
      • 이름 : 송장에 표시할 수 있는 쿠폰의 표시 이름
      • 기간 : 선택한번 , 영원히또는반복 ;
        만약반복선택한 다음 반복적으로 사용할 수있는 개월 수를 설정합니다.
      • 할인 유형에대해 다음 중 하나를 선택합니다.
        • 화폐 금액 : 쿠폰의 값 및 통화를 입력합니다.
        • 백분율 : 쿠폰이 운반할 할인의 비율을 입력하십시오.

        예: 할인이 3개월 무료인 경우기간을 3개월로반복으로설정하고할인 유형을백분율 100% 로 설정합니다.

      • 제한 날짜 (UTC) : 쿠폰을 사용할 수 있는 마지막 날짜를 선택하세요
      • 최대 사용 : 쿠폰의 최대 사용 횟수를 입력합니다. 예를 들어 50명의 고객만 쿠폰을 사용할 수 있습니다.
      • 패키지 선택 : 쿠폰을 적용할 수 있는 패키지를 선택합니다. 패키지를 선택하지 않으면 쿠폰을 모든 패키지에 적용할 수 있습니다.
        예: 쿠폰은 연간 구독을 구매하는 경우에만 유효합니다.

    3. 새 쿠폰을저장하려면저장 버튼을 클릭하십시오.

    패키지 구매가 이루어지면 브라이트코브 비콘 앱에서 시청자는 다음과 같이 쿠폰ID를입력할 수 있습니다.

    또는 Stripe 계정에서 직접 쿠폰을 생성한 다음 브라이트코브 비콘에서 사용할 수 있습니다.
