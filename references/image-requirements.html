비디오 클라우드 이미지

비디오가 Video Cloud로 수집될 때마다 두 개의 이미지가 자동으로 생성되어 비디오 속성에 표시됩니다. 생성된 두 개의 이미지는 포스터 그리고 썸네일. 이미지는이미지섹션을 참조하세요. Brightcove Beacon에서 사용하기 위해 이러한 업로드 이미지를 사용하지만 적절한 기능을 위해서는 기본 수집 프로필로 변경해야 합니다. 다음 경고 상자에 표시된 단계는 변경 방법을 보여줍니다. 변경하지 않으면 업로드하는 세로 이미지의 크기가 계속 축소판에 맞게 조정되고 Brightcove Beacon 앱에서 매우 좋지 않게 보입니다.

기본 수집 프로필을 변경해야 합니다. 이 차례를 따라라:
  1. 스튜디오에서 이동관리자 → 프로필 수집 .
  2. 로 레이블이 지정된 현재 기본 프로필의 제목을 클릭합니다. 업로드 기본값상표.
  3. 클릭복제하다단추.
  4. 위치 찾기이미지정의됩니다. 정확한 JSON은 복제한 프로필에 따라 아래 표시된 것과 약간 다르게 보일 수 있습니다.
  5. 다음과 같이 포스터 및 축소판 이미지의 너비와 높이를 변경합니다.

    이전 설정

    {
  "media_type": "image",
  "format": "jpg",
  "label": "poster",
  "width": 1280,
  "height": 720,
  "id": "559a8b0ee4b07cbb40be7d22"
},
{
  "media_type": "image",
  "format": "jpg",
  "label": "thumbnail",
  "width": 160,
  "height": 90,
  "id": "559a8b0ee4b07cbb40be7d23"
}

    새 설정

    {
  "media_type": "image",
  "format": "jpg",
  "label": "poster",
  "width": 4000,
  "height": 2250,
  "id": "559a8b0ee4b07cbb40be7d22"
},
{
    "media_type": "image",
    "format": "jpg",
    "label": "thumbnail",
    "width": 3000,
    "height": 2000,
    "id": "559a8b0ee4b07cbb40be7d23"
}
  6. 딸깍 하는 소리구하다 .
  7. 그런 다음내 기본 프로필로 설정링크.
  8. 이제 프로필에 업로드 기본값으로 레이블이 지정되었는지 확인하세요.
이제 Video Cloud Studio를 사용하여 Brightcove Beacon용 이미지를 업로드할 준비가 되었습니다.

Studio 인터페이스를 사용하여 두 개의 이미지를 업로드합니다. 비디오를 나타내는 이미지를 결정하고 위에 표시된 요구 사항을 충족하는 적절한 스크린샷을 찍습니다. 그런 다음 두 이미지를 Video Cloud에 업로드합니다.

NS 포스터 그리고 썸네일 Brightcove Beacon에서 사용되는 이미지의 자리 표시자입니다. Video Cloud Studio에 표시된 이미지에 대한 권장 사항을 따르지 마십시오! 대신 다음 지침을 사용하십시오.

포스터 썸네일
풍경 이미지(16:9):
최저한의 너비: 1000픽셀 키: 563픽셀
최고 너비: 4000픽셀 키: 2250픽셀
포스터 이미지(2:3)
최저한의 너비: 1000픽셀 키: 1500픽셀
최고 너비: 2000픽셀 키: 3000픽셀

스크린샷의 왼쪽 이미지는 세로 이미지의 사용 예입니다. 매우 픽셀화된 이미지가 보이면 기본 수집 프로필을 업데이트해야 한다는 것을 알 수 있습니다.

기업 로고

Brightcove Beacon Experience에서 사용하려면 회사 로고를 제공해야 합니다. 이미지에 대해 다음 지침을 따르십시오.

사양 값
체재 .png
종횡비 1:1
크기 1500픽셀 x 1500픽셀
