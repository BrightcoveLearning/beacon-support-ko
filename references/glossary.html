개요

다음 플랫폼에 대한 고객의 결제를 허용하려면 Stripe 계정이 필요합니다.

  • 편물
  • 삼성 스마트 TV
  • 소니 스마트 TV
  • 도시바 스마트 TV

이는 웹 앱과 모든 스마트 TV 앱에 Stripe가 필요함을 의미합니다. 이것 말고도 안드로이드 TV와 파이어 TV.

다음 단계는 Brightcove Beacon과 함께 Stripe를 사용하는 프로세스에 대한 개요를 제공합니다.

  1. 기본 정보(이메일, 이름, 비밀번호)를 제공하여 계정을 만듭니다.
  2. 귀하의 비즈니스에 대한 정보를 제공하여 계정을 활성화하십시오.
  3. Stripe API 키를 검색합니다.
  4. Brightcove 직원에게 API 키를 제공하여 그들이 Brightcove Beacon Experience에 추가할 수 있도록 합니다.

Stripe 계정 생성 및 활성화

  1. 찾아보기스트라이프 홈페이지 .
  2. 클릭지금 시작단추.
  3. 기본 정보를 입력하여 계정을 만듭니다.
  4. Stripe 대시보드가 표시되었는지 확인하십시오.
  5. 클릭 Stripe 계정 활성화옵션.
  6. 클릭지금 시작버튼을 눌러 프로세스를 시작합니다.
  7. 이메일 주소를 확인해야 합니다.
  8. 이메일 주소를 확인하면 계정 활성화를 위해 작성해야 하는 양식이 표시됩니다. 노란색으로 강조 표시된 부분은 양식의 주요 부분입니다.
  9. 양식 작성 후지원서 제출단추.

Stripe API 키 가져오기

Stripe 계정과 Brightcove Beacon 간의 연결은 Stripe API 키를 통해 이루어집니다. 위에 표시된 대시보드의 스크린샷에서 다음 링크를 볼 수 있습니다. 라이브 API 키 가져오기 . 계정이 활성화될 때까지는 테스트 API 키만 받을 수 있습니다.

대시보드에서라이브 API 키 가져오기지침에 따라 다운로드하십시오.

Brightcove Beacon에 Stripe 연결하기

Stripe API 키가 있으면 Brightcove Beacon Experience를 구성하는 Brightcove 직원에게 제공합니다. Stripe API 키를 Brightcove Beacon Experience와 연결하여 앱에서 사용하도록 Stripe를 구성할 때 계정과 연결되도록 합니다.
