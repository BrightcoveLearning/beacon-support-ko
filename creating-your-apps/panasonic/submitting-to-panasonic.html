소개

Brightcove Beacon을 사용하면 Panasonic Store에 추가할 수 있는 생성된 장치 앱을 받게 됩니다.

  • 매장 정보: https://developer.vieraconnect.com/
  • 승인을 위한 제안된 리드 타임: 3주

Panasonic에 필요한 정보:

  • 애플리케이션
    • 애플리케이션 URL 및 애플리케이션 스테이징 URL
  • 설명
    • 최대 1024자
    • 최대 26줄
    • 각 줄은 최대 85자(또는 아시아 44자)로 구성됩니다.
    • 적절한 곳에 줄 바꿈을 삽입하십시오.
  • 아이콘
    • 4개의 아이콘 이미지가 필요합니다.
      • 160x112(PNG)
      • 배너 스타일 아이콘 256x60(PNG)
      • 360x252(PNG)
      • 원형 원형 아이콘 지름 336x336(PNG)
  • 스크린샷
    • 적어도 하나는 필요합니다
    • 480w x 270h PNG
    • 마켓의 애플리케이션 세부정보 페이지에 표시됩니다.

시작하기

Panasonic 개발자 페이지에 액세스하려면 다음 단계를 따르세요.

  1. 브라우저에서파나소닉 개발자 로그인페이지.

  2. 계정으로 로그인하거나 새 계정을 만드십시오.

  3. 로그인하면 제어판이 표시됩니다.

  4. 오른쪽에 있는 메모를 읽으십시오. 새 신청서 제출단추. 선택홈페이지링크.

  5. 앱 요구 사항을 검토합니다. 걱정하지 마세요. Brightcove가 필요한 기술로 앱을 만들 것입니다.

신청 정보

NS 신청 정보섹션은 기본 앱 정보를 입력하는 곳입니다.

  1. 제어판에서 다음을 선택합니다. 새 신청서 제출단추.

  2. 다음과 같은 입력 양식이 표시되어야 합니다.

  3. 에서신청 정보페이지에서 필수 필드에 대한 값을 입력합니다.

    • 프로젝트 유형 - 드롭다운 메뉴에서 사용 가능한 최신 HTML 버전을 선택합니다.
    • 신청 URL - Brightcove에서 앱 빌드를 호스팅할 두 개의 리포지토리를 만듭니다. 하나는 프로덕션용이고 하나는 스테이징용입니다. 애플리케이션 URL은 프로덕션 URL입니다.
    • 스테이징 URL - 앱 빌드의 스테이징 URL을 입력합니다.
    • 버전 번호 - 앱 버전을 입력합니다.

  4. 선택추가/제거단추. 앱에 사용할 언어를 선택합니다. 선택확인하다단추.

  5. 선택한 언어로 필드 값을 입력합니다. 이 예에서는영어(북미)선택됩니다.

    • 기본 설정 언어 - 드롭다운 메뉴에서 앱의 초기 언어를 선택합니다. 앱이 영어를 지원하는 경우 Panasonic은 영어((북미) 또는 영어(영국 모델))를 선택할 것을 권장합니다.
    • 애플리케이션 이름 - 앱에 사용할 이름을 입력합니다.
    • 짧은 이름 - 앱에 사용할 짧은 이름을 입력합니다. 애플리케이션 이름과 동일할 수 있습니다.
    • 키워드 - 마켓 검색 기능에서 사용할 키워드를 입력하여 앱을 찾습니다. 쉼표로 키워드를 구분하십시오(예: 탁구, 스포츠). 최대 128자입니다.
    • 설명 - 앱에 대한 설명을 입력합니다. 이는 마켓의 애플리케이션 세부 정보 페이지에 표시됩니다. 자세한 내용을 보려면 입력 상자 내부를 클릭하십시오.
    • 최종 사용자 라이선스 계약 - 선택사항. 최종 사용자 라이선스 계약을 입력합니다.

  6. 선택한 언어 목록에서 다음 언어를 선택하고 이전 단계에서 설명한 필드 값을 입력합니다.

  7. 선택다음단추.

추가 세부 사항

NS 추가 세부 사항섹션에서 지원 및 콘텐츠 세부 정보를 추가할 수 있습니다.

  1. 선택 후다음버튼을 누르면 다음과 같은 입력 양식이 표시됩니다.

  2. 에서추가 세부 사항페이지에서 필수 필드에 대한 값을 입력합니다.

    • 지원 이메일 - 사용자가 앱에 문제가 발생했을 때 연락할 수 있는 이메일 주소를 제공합니다.
    • 부모 등급 - 드롭다운 메뉴를 확장하여 각 자녀 보호 등급에 대한 정의를 확인합니다. 앱에 적합한 등급을 선택하세요.
    • 브라질 콘텐츠 등급 - 브라질 출시 예정인 경우 필수 항목입니다. 드롭다운 메뉴를 확장하여 각 브라질 등급에 대한 정의를 확인합니다. 적절한 등급을 선택하십시오.
    • 이메일 연락처 - 연락처 이메일 주소를 입력합니다.
    • 범주 - 드롭다운 메뉴를 확장하여 카테고리 목록을 봅니다. 적절한 카테고리를 선택하십시오.
    • 가격 - 일반적으로 Panasonic의 앱은 무료로 다운로드할 수 있습니다.
    • 인앱 구매 - 앱에 인앱 구매가 포함되는지 여부를 선택합니다.
    • 마켓플레이스 - 클릭마켓플레이스 선택버튼을 눌러 국가 목록을 봅니다. 선택하거나모든 나라들또는 앱을 사용할 수 있는 국가를 선택합니다. 완료되면 선택완료단추.
    • 품질보증을 위한 지침 - 버튼 추가, 링크 변경 등을 포함하여 도움이 될 수 있는 지침을 포함합니다. 필요한 경우 앱에 대한 테스트 계정을 포함합니다.

  3. 선택다음단추.

자산 업로드

NS 자산 업로드섹션에서는 아이콘과 이미지를 추가할 수 있습니다.

  1. 선택 후다음버튼을 누르면 다음과 같은 입력 양식이 표시됩니다.

  2. 에서자산 업로드페이지에서 필수 필드에 대한 값을 입력합니다.

    • 앱 설명 문서 - 앱 작동 방식을 자세히 설명하는 문서를 만듭니다. 이것은 LG용으로 생성한 것과 동일한 문서일 수 있습니다. 선택검색버튼을 눌러 파일을 찾습니다. 그런 다음업로드단추.

      참고로doc , docx또는pdf파일을 업로드할 수 있습니다. 최대 파일 크기는 8MB입니다.

    • 아이콘 1 - 에 표시될 마켓 아이콘을 업로드합니다. 목록 페이지그리고신청 상세 페이지마트에서.

      160w x 112h PNG

    • 아이콘 2(원형) - Firefox OS 아이콘 업로드앱 문서 페이지마트에서.

      336 픽셀 지름의 원형 모양 PNG

    • 샘플 이미지 1 - 다음은 앱의 스크린샷이며 화면에 표시됩니다. 신청 상세 페이지마트에서. 최대 5개의 샘플 이미지를 추가합니다.

      480w x 270h PNG

  3. 선택다음단추.

T&C에 동의

NS T&C에 동의섹션에서는 스토어에 제출하기 전에 앱에 추가한 모든 정보를 검토할 수 있습니다. Panasonic 이용 약관이 있는 섹션도 있습니다.

  1. 선택 후다음버튼을 누르면 다음과 같은 입력 양식이 표시됩니다.

  2. 앱에 추가한 정보와 이미지를 검토합니다.

  3. Panasonic TV 이용약관을 읽어보십시오. 읽었으며 동의함을 나타내는 확인란을 선택하십시오.

  4. 선택제출하다단추.

  5. 에애플리케이션 목록페이지에서 앱이 초안으로 저장된 것을 볼 수 있습니다.

  6. 앱이 출시되기 전에 QA 상태가 됩니다.
