    파나소닉 스토어에 앱 제출

    이 항목에서는 기기 앱을 Panasonic 스토어에 제출하는 방법을 배우게 됩니다.

    서문

    브라이트코브 비콘을 사용하면 생성된 디바이스 앱을 Panasonic 스토어에 추가할 수 있습니다.

    • 매장 상세 정보: https://developer.vieraconnect.com/
    • 승인을 위한 제안된 리드 타임: 쓰리 주

    파나소닉에 필요한 정보:

    • 응용 프로그램
      • 응용 프로그램 URL 및 응용 프로그램 준비 URL
    • 설명
      • 최대 1024자
      • 최대 26 라인
      • 최대 85자 (또는 아시아 문자 44자) 로 구성된 각 행
      • 적절한 경우 줄 바꿈을 삽입하십시오.
    • 아이콘
      • 네 개의 아이콘 이미지가 필요합니다.
        • 피크스버그
        • 배너 스타일 아이콘 256x60 (PNG)
        • 360x252 (PNG)
        • 원형 원형 아이콘 지름 336x336 (PNG)
    • 스크린샷
      • 적어도 하나가 필요합니다.
      • 최대 480세로 x 270시간
      • 이러한 정보는 마켓의 애플리케이션 세부 정보 페이지에 표시됩니다.

    시작하기

    Panasonic 개발자 페이지에 액세스하려면 다음과 같이 하십시오.

    1. 브라우저에서Panasonic 개발자 로그인페이지로 이동합니다.

    2. 계정으로 로그인하거나 새 계정을 만듭니다.

    3. 로그인하면 제어판이 표시됩니다.

    4. 새 신청서 제출버튼 오른쪽에 있는 노트를 읽습니다. 홈 페이지링크를 선택합니다.

    5. 앱 요구 사항을 검토합니다. 걱정하지 마세요. 브라이트코브는 필요한 기술로 앱을 제작할 것입니다.

    응용 프로그램 정보

    응용 프로그램 정보섹션은 기본 앱 정보를 입력하는 곳입니다.

    1. 제어판에서새 응용 프로그램 제출단추를 선택합니다.

    2. 다음과 같은 입력 양식이 표시되어야합니다.

    3. [ 응용 프로그램 정보 ] 페이지에서 필수 필드에 값을 입력합니다.

      • 프로젝트 유형 - 드롭다운 메뉴에서 사용 가능한 최신 HTML 버전을 선택합니다.
      • 애플리케이션 URL - 브라이트코브에서 앱 빌드를 호스팅할 리포지토리를 두 개 만듭니다. 하나는 프로덕션 용이고 다른 하나는 스테이징 용입니다. 응용 프로그램 URL은 프로덕션 URL입니다.
      • 준비 URL - 앱 빌드의 준비 URL을 입력합니다.
      • 버전 번호 - 앱 버전을 입력합니다.

    4. 추가/제거버튼을 선택합니다. 앱에 사용할 수 있는 언어를 선택합니다. 확인버튼을 선택합니다.

    5. 선택한 언어로 필드에 대한 값을 입력합니다. 이 예에서는영어 (북미)가 선택됩니다.

      • 기본 언어 - 드롭다운 메뉴에서 앱의 초기 언어를 선택합니다. 앱이 영어를 지원하는 경우 파나소닉에서는 영어 (북미) 또는 영어 (영국 모델) 를 선택하는 것이 좋습니다.
      • 응용 프로그램 이름 - 앱에 사용할 이름을 입력합니다.
      • 약어 이름 - 앱에 사용할 약어 이름을 입력합니다. 응용 프로그램 이름과 같을 수 있습니다.
      • 키워드 - 마켓 검색 기능에서 앱을 찾는 데 사용할 키워드를 입력합니다. 키워드는 쉼표로 구분합니다 (예: 탁구, 스포츠). 최대 128자.
      • 설명 - 앱에 대한 설명을 입력합니다. 이는 Market의 애플리케이션 세부 정보 페이지에 표시됩니다. 자세한 내용을 보려면 입력 상자 내부를 클릭하십시오.
      • 최종 사용자 사용권 계약 - 선택 사항 최종 사용자 사용권 계약을 입력합니다.

    6. 선택한 언어 목록에서 다음 언어를 선택하고 이전 단계에서 설명한 필드 값을 입력합니다.

    7. 다음버튼을 선택합니다.

    추가 세부 정보

    추가 세부 정보섹션에서는 지원 및 컨텐츠 세부 정보를 추가할 수 있습니다.

    1. 이전 섹션에서다음버튼을 선택하면 다음과 같은 입력 양식이 표시됩니다.

    2. 추가 상세내역페이지에서 필수 필드에 대한 값을 입력합니다.

      • 지원 이메일 - 사용자가 앱에 문제가 발생할 때 연락할 수 있는 이메일 주소를 제공합니다.
      • 보호자 등급 - 드롭다운 메뉴를 확장하여 각 보호자 등급에 대한 정의를 확인합니다. 앱에 적합한 등급을 선택합니다.
      • 브라질 콘텐츠 등급 - 브라질에서 릴리스하려는 경우 필요합니다. 드롭다운 메뉴를 확장하여 각 브라질 등급에 대한 정의를 확인합니다. 적절한 등급을 선택합니다.
      • 연락처 이메일 - 연락처 이메일 주소를 입력합니다.
      • 범주 - 범주목록을 보려면 드롭 다운 메뉴를 확장합니다. 적절한 범주를 선택합니다.
      • 가격 - 일반적으로 Panasonic의 앱은 무료로 다운로드 할 수 있습니다.
      • 인앱 구매 - 앱에 인앱 구매를 포함할지 여부를 선택합니다.
      • 시장 장소 - 국가 목록을 보려면시장 장소 선택버튼을 클릭합니다. 모든 국가를선택하거나 앱을 사용할 수 있도록 하려는 국가를 선택합니다. 완료되면완료버튼을 선택합니다.
      • QA 지침 - 버튼 추가, 링크 변경 등 도움이 될 수 있는 모든 지침을 포함합니다. 필요한 경우 앱에 대한 테스트 계정을 포함합니다.

    3. 다음버튼을 선택합니다.

    자산 업로드

    자산 업로드섹션에서 아이콘과 이미지를 추가할 수 있습니다.

    1. 이전 섹션에서다음버튼을 선택하면 다음과 같은 입력 양식이 표시됩니다.

    2. 자산 업로드페이지에서 필수 필드에 값을 입력합니다.

      • 앱 설명 문서 - 앱의 작동 방식을 자세히 설명하는 문서를 만듭니다. 이 문서는 LG 용으로 만든 문서와 동일할 수 있습니다. 찾아보기버튼을 선택하여 파일을 찾습니다. 그런 다음업로드버튼을 선택합니다.

        참고doc , docx또는pdf파일 업로드가 허용됩니다. 최대 파일 크기는 8MB입니다.

      • 아이콘 1 - 마켓의리스팅 페이지및애플리케이션 상세 페이지에표시되는 마켓 업로드 아이콘

        제품 공급 시 112시간

      • 아이콘 2 (원형 모양) - 파이어 폭스 OS 아이콘을 업로드합니다. 이 아이콘은 시장의앱 문서 페이지에표시됩니다.

        336 픽셀 직경 원 모양 PNG

      • 샘플 이미지 1 - 앱의 스크린 샷으로 시장의애플리케이션 세부 정보 페이지에표시됩니다. 최대 5개의 샘플 이미지를 추가할 수 있습니다.

        최대 480세로 x 270시간

    3. 다음버튼을 선택합니다.

    이용 약관 동의

    이용 약관 동의섹션에서 스토어에 제출하기 전에 앱에 추가한 모든 정보를 검토할 수 있습니다. Panasonic 이용 약관 섹션도 있습니다.

    1. 이전 섹션에서다음버튼을 선택하면 다음과 같은 입력 양식이 표시됩니다.

    2. 앱에 추가한 정보와 이미지를 검토합니다.

    3. 파나소닉 TV에 대한 이용 약관을 읽어보십시오. 읽고 동의한 상자를 선택하십시오.

    4. 실행버튼을 선택합니다.

    5. 응용 프로그램 목록페이지에서 초안으로 저장된 앱이 표시됩니다.

    6. 앱이 라이브로 전환되기 전에 QA 상태로 전환됩니다.
