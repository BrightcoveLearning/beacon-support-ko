페이지 내용

    비디오 클라우드 사용자 정의 필드 만들기

    이 항목에서는 브라이트코브 비콘과의 동기화에 필요한 Video Cloud 사용자 정의 필드를 볼 수 있습니다.

    서문

    Video Cloud에서 브라이트코브 비콘으로 가져올 동영상은 브라이트코브 비콘에서 적절하게 사용할 수 있도록 정의된 풍부한 메타데이터 세트가 있어야 합니다. 이 문서에서는 Studio에서 새 사용자 정의 필드를 만드는 방법을 볼 수 있습니다. 또한 브라이트코브 비콘에 필요한 필드가 정의됩니다.

    사용자 정의 필드 만들기

    사용자 지정 필드를 만들려면 다음과 같이 하십시오.

    1. 비디오 클라우드 스튜디오에 로그인합니다.
    2. 탐색 헤더에서ADMIN을클릭한 다음비디오 필드를클릭합니다. ADMIN 메뉴에 액세스하려면 계정 관리자여야 합니다.
    3. 다음 스크린샷과 같이 비디오 필드 페이지에는 사용자 정의 필드와 모든 표준 비디오 메타데이터 필드가 표시됩니다.
    4. 사용자 정의 필드 추가를클릭합니다.

    5. 표시 이름을입력합니다. 업로드 및 미디어 모듈의 필드에 대한 레이블로 나타납니다. 표시이름을 기반으로 필드에 대한 내부이름이 자동으로 생성됩니다.

      참고: 브라이트코브 비콘으로 성공적으로 가져오기 위해 사용할 표시 이름은 모두 이 문서의 뒷부분에 명시되어 있습니다.
    6. 필드에 대해유형을선택합니다.
      • 텍스트 - 사용자는 최대 1024개의 싱글바이트 문자 (또는 512개의 더블바이트 문자) 까지 필드에 아무 것도 입력할 수 있습니다.
      • 목록 - 사용자가 값 목록에서 선택합니다. 목록에 값으로 150개 이하를 입력할 수 있으며, 목록의 각 값은 100자 길이 이내로 입력할 수 있습니다.
    7. 설명을입력합니다. 설명 필드는 Studio 인터페이스에 힌트로 나타납니다.
    8. 사용자 정의 필드가 필요한 경우이 필드를 필수로 만들기를선택합니다.

    9. 필드 유형이목록인 경우, 값의 쉼표로 구분 된 목록을 입력 한 다음 다음 다음 스크린 샷과 같이목록에 추가를클릭합니다.

      참고:
      1. 값에는 따옴표가 포함되지 않아야 합니다 (예: 동물, 새, 물고기).
      2. 값은 쉼표를 포함 할 수 없습니다. 모든 쉼표는 서로 다른 값의 구분 기호로 처리됩니다.
    10. 필드 저장을클릭하여 변경 사항을 저장합니다. 저장 및 다른 항목 추가를클릭하여 필드를 저장하고 다른 필드 만들기를 시작합니다.

    작성한 사용자 정의 필드는 즉시 사용할 수 있으며 미디어 모듈에 표시됩니다  .

    최대 10개의 사용자 정의 필드를 만들 수 있습니다. 10개 이상의 필드가 필요한 경우추가 필드를 클릭합니다.링크를 클릭하여 브라이트코브 고객 지원 센터에 문의하여 자세한 내용을 요청하십시오. 게시자는 50개의 사용자 지정 필드로 제한됩니다.

    목록 값 편집

    목록 유형 사용자 정의 필드에 값을 추가할 수 있습니다. 사용자 정의 필드 이름 링크를 클릭하고 값을 추가한 다음목록에 추가를클릭합니다. 완료되면필드 저장을클릭합니다.

    목록 값 삭제

    목록 유형 사용자 정의 필드에서 기존 값을 제거할 수 있습니다. 사용자 정의 필드 이름 링크를 클릭하고 값과 연관된 삭제 아이콘 ( ) 을 클릭한 다음필드 저장을클릭합니다.

    정가 제거되면 이전에 해당 값이 할당되었던 모든 동영상은 값을 유지합니다. 그러나 삭제 된 값이있는 비디오를 편집하면 값이 지워지고 새 값을 선택해야합니다.

    계정에서 사용자 정의 필드 제거

    계정에서 사용자 지정 필드를 제거하려면브라이트코브 지원에 문의해야합니다. 브라이트코브 지원에서 맞춤 필드를 삭제한 경우 값이 있는 동영상의 수정 날짜가 변경됩니다.

    브라이트코브 비콘 메타데이터에 대한 목록 값 사용

    아래 표에서 생성해야 하는 여러 사용자 정의 필드를 찾을 수 있습니다. 위에 표시된 것처럼 사용자 정의 필드를 만들 때 데이터 유형을텍스트또는목록으로선택합니다. 모든유형은텍스트입니다 . 텍스트를 입력할 필요 없이 목록에서 선택할 수 있다면 동영상 콘텐츠 큐레이터의 부담을 덜어줄 수 있습니다. 또한 값을 입력하지 않고 목록에서 선택할 수 있으면 데이터 입력을보다 일관되게 유지하는 데 도움이됩니다. 따라서List가일부 비즈니스 사례에서 작동하는 경우 목록을 사용할 수 있습니다.

    물론 선택은 귀하의 것이지만 일부 사용자 정의 필드의 경우List가데이터 유형에 적합한 선택이라는 것을 알 수 있습니다. 예를 들어, 목록에 대해 다음을 고려할 수 있습니다.

    • 비콘 장르
    • 비콘. 응축하는
    • 비콘. 뷰어스코어
    • 비콘. 권리. 유형

    표준 필드

    다음 표에는 표준 Video Cloud 필드가 표시되어 있습니다. 즉, 이 필드는 기본적으로 존재하며, 브라이트코브 비콘에서 메타데이터가 연결된 비디오를 올바르게 가져오는 데 사용됩니다.

    문자 제한에 대한 위의 6 단계에서 간단한 알림:

    • 텍스트 - 최대 1024개의 싱글바이트 문자 (또는 512개의 더블바이트 문자) 를 포함할 수 있습니다.
    • 목록 - 150 개 이하의 가능한 값을 가질 수 있으며 목록의 각 값은 100 자 이하일 수 없습니다.
    참고: 스튜디오 UI에서 요청한 이미지와 다른 이미지를 제공하므로이미지.poster.src 및 images.thumbnail.src의정의에 주의하십시오. 여기에 표시된 UI를 사용하여 이미지를 추가하지만 UI가 아닌 표의 지침에 따라 이미지를 추가합니다.

    사용자 정의 필드 이름

    이 문서의 다음 세 섹션에서는 특정 구현 사용 사례를 기반으로 만들어야 할 수 있는 사용자 정의 필드를 다룹니다. 각 사용자 지정 필드에는표시 이름과내부 이름이모두 있습니다. 표시 이름 옵션에 대해 더 의미 있는 이름이 있다고 생각되면 해당 이름을사용할 수 있지만내부 이름은정확히MUST일치합니다.

    권한 관리 필드

    자산에 대한 수익 창출 및 액세스 제한을 지원하려면 일련의 사용자 지정 필드를 사용해야 합니다. 권한 관리 구성의 인스턴스를 여러 개 가질 수 있으므로 여러 개의 사용자 지정 필드 집합을 만드는 것이 좋습니다. 각 인스턴스의 이름에서 카운터는 그것들을 구별하는 데 사용됩니다. 첫 번째 인스턴스는 카운터를 0으로 사용해야합니다. 다음은 실제로 카운터 값을 사용하는 그룹화 된 필드의 예입니다.

    예를 들어 동영상에 사용되는 광고를 정의하기 위한 집합이 필요하고 새해 같은 휴일에 대한 특별 광고를 정의하는 집합이 필요할 수 있습니다.

    구체적인 예는 이러한 필드의 사용을 명확히하는 데 도움이됩니다. 다음 스크린샷은 Video Cloud의 비디오에 사용되는 서버 측 광고 삽입 (SSAI를) 설정을 보여줍니다.

    이 예제에서 다음을 배울 수 있습니다.

    • 그룹의 모든 권한 관리 필드에 값을 할당해야 하는 것은 아닙니다. 이 경우, 단지비콘. 권리.<카운터 >.AdConfiguration 및 신호.권한.< >카운터. type이사용됩니다.
    • 브라이트코브 비콘 사용자 정의 필드에서 Studio에서 만든 SSAI구성 ID를 사용할 수 있습니다.
    • 권한 관리 필드 집합에 대한 카운터 값은 0 (노란색으로 강조 표시됨) 에서 시작할 수 있습니다.
    • 이 스크린샷에서 직접 볼 수는 없지만 사실, Video Cloud SSAI의 구성이 브라이트코브 비콘으로 수집된다는 것입니다.

    작성하는 필드는이 필드를 필수로 만들기확인란을 선택할 필요가 없습니다.
