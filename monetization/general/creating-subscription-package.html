구독 패키지 만들기

이 주제에서는 다음을 사용하여 비디오 구독 패키지를 구성하는 방법을 배웁니다. 상업 Brightcove Beacon 탭. 구독 패키지는 SVOD(주문형 구독 비디오) 수익 창출을 구현합니다. 동영상에 구독 패키지를 적용하는 방법을 알아보려면구독 패키지 사용문서.

개요

구독 패키지는 앱에서 특정 비디오 세트를 구매할 수 있는 방법을 정의합니다.' 고객. 예를 들어 패키지를 생성할 때 다음을 구성할 수 있습니다.

  • 청구 빈도
  • 광고 지원 여부 및 사용할 광고 구성 설정
  • 지불 방법
  • 패키지 사용자가 동시에 재생할 수 있는 최대 스트림 수
  • 패키지를 구매할 수 있는 시작 및 종료 날짜
  • 패키지의 비디오를 볼 수 있도록 허용된 장치 및 지리적 위치

패키지를 만드는 단계

새 패키지를 생성하려면 다음 단계를 따르십시오.

  1. Brightcove Beacon에서상업탭, 확인패키지왼쪽 탐색에서 을 선택한 다음새 패키지 추가 .
  2. 에패키지탭(아래 스크린샷)에서 다음을 완료합니다.
    • 이름 : 비디오의 Video Cloud 사용자 정의 필드에서 사용되는 패키지의 이름입니다.
    • 청구 빈도 : 이 패키지의 청구 빈도를 선택하십시오.
    • 평가판 기간 : 이 패키지의 평가판 기간입니다. 입력하다 0평가판 기간이 없습니다.
    • 상태 : 패키지를 게시하여 사용할 수 있도록 하려면 선택합니다. 하나 이상의 결제 방법을 선택해야 패키지를 게시할 수 있습니다.
    • 광고 : 이 패키지와 함께 광고를 사용하려면 선택하십시오.
    • 광고 제공업체 : 이전 단계에서 광고를 사용하기로 선택한 경우 광고 제공업체를 선택합니다. 이 드롭다운은 목록에 나열된 광고 구성으로 채워집니다. 광고 Brightcove Beacon 섹션.
    • 사용할 결제 수단 : 이 패키지에 사용할 결제 수단을 선택하세요. 선택한 각 방법에 대해 섹션이 결제 방법 목록 아래에 열리며 선택한 각 방법에 대해 완료해야 합니다. 아래 스크린샷은 보여줍니다스트라이프(노란색)그리고애플 스토어(빨간색)선택됩니다.
  3. 클릭패키지 생성버튼을 눌러 현재 패키지 구성을 저장합니다.
  4. 에텍스트 데이터탭에서 다음을 완료합니다.
    • 팩 이름 : 사용자에게 표시되는 이름입니다.
    • 표제 : 예를 들어 사용자에게 표시되는 헤드라인 홀리데이 스페셜! .
  5. 클릭패키지 생성버튼을 눌러 현재 패키지 구성을 저장합니다.
  6. 에비텍스트 데이터탭에서 패키지 이름 옆에 사용할 이미지를 제공합니다.
  7. 클릭패키지 생성버튼을 눌러 현재 패키지 구성을 저장합니다.
  8. NS 콘텐츠탭은 현재 사용하지 않습니다.
  9. 에스트림탭에서 다음을 완료합니다.
    • 최대 스트림 : 동시에 스트리밍할 수 있는 최대 연결된 장치 수입니다.
    • 최대 장치 : 활성 연결을 가질 수 있는 연결된 장치의 최대 수입니다.
      위의 두 가지 설정도 전역적으로 설정되므로 이 패키지의 전역 설정을 무시하려는 경우에만 이 설정을 사용하십시오.
    • 고급 스트림 : 다음으로 설정예이 패키지가 설정하는 권한을 사용자 정의하려면 사용 가능한 변경 사항은 다음 스크린샷에 표시됩니다.
  10. 클릭새 권한 추가버튼을 눌러 새로 구성된 사용자 정의 권한을 저장합니다.
  11. 클릭패키지 생성버튼을 눌러 현재 패키지 구성을 저장합니다.

그것이 바로 패키지 생성을 위한 것입니다!

스트라이프 쿠폰 만들기

쿠폰을 생성하여 패키지를 구매할 때 적용할 수 있는 고정 금전적 할인 또는 백분율 할인을 제공할 수 있습니다.

  1. Brightcove Beacon에서상업탭을 클릭한 다음스트라이프 쿠폰그리고 마지막으로 클릭새 바우처 추가 .
  2. 에새 쿠폰 만들기양식(아래 스크린샷)을 작성하여 다음을 완료하십시오.
    • ID : 시청자가 쿠폰을 사용하기 위해 Brightcove Beacon 앱에 입력하는 코드
    • 이름 : 인보이스에 표시할 수 있는 쿠폰의 친숙한 이름
    • 지속 : 에서 선택한 번 , 영원히또는반복 ;
      만약에반복을 선택한 다음 반복적으로 사용할 수 있는 개월 수를 설정합니다.
    • 을위한할인 유형다음 중 하나를 선택합니다.
      • 금액 : 쿠폰의 가치와 통화를 입력하십시오.
      • 백분율 : 쿠폰에 적용되는 할인 비율을 입력하세요.

      예시: 할인이 3개월 무료인 경우 다음을 설정합니다. 지속에게반복 3개월로 설정하고할인 유형 ~와 함께백분율 100%

    • 제한 날짜(UTC) : 쿠폰을 사용할 수 있는 마지막 날짜를 선택하세요.
    • 최대 상환 : 쿠폰의 최대 상환 횟수를 입력하십시오. 예를 들어 50명의 고객만 쿠폰을 사용할 수 있습니다.
    • 패키지 선택 : 쿠폰을 적용할 수 있는 패키지를 선택하십시오. 패키지를 선택하지 않으면 모든 패키지에 쿠폰 적용 가능
      예시: 쿠폰은 연간 구독을 구매한 경우에만 유효합니다.

  3. 반드시 클릭하세요구하다버튼을 눌러 새 쿠폰을 저장합니다.

패키지 구매 시 Brightcove Beacon 앱에서 시청자가 쿠폰을 입력할 수 있습니다. ID :

또는 Stripe 계정에서 직접 쿠폰을 생성한 다음 Brightcove Beacon에서 사용할 수 있습니다.
