소개

Evergent 백오피스 시스템은 비즈니스 관리에 도움이 되는 서비스를 제공합니다. 이 주제에서는 이러한 서비스 중 두 가지를 다룹니다.

  • 자동화된 이메일
  • 쿠폰 관리자

이메일 발송자

전자 메일러를 사용하면 다양한 시스템 생성 전자 메일에 대한 템플릿과 일정을 만들 수 있습니다. 다음 섹션에서는 전자 메일러 작업에 대해 자세히 설명합니다.

메시지 유형

Evergent는 다음 메시지 유형을 지원합니다.

스케줄링
시스템 관리자에서 일괄 작업이 예약되면 관리자에게 이메일이 전송됩니다.
신용 카드 만료 알림
비즈니스 수준에서 "신용 카드 만료 알림" 확인란을 선택하면 신용 카드가 곧 만료될 때 알림 이메일이 고객에게 전송됩니다(이메일은 일반적으로 카드 만료 한 달 전에 전송됨). 이 경우 시스템은 이메일 템플릿 메시지 유형 "신용카드 만료 알림"을 사용합니다.
대량 이메일
비즈니스에서 프로모션 이메일을 예약하고 실행할 향후 시간을 지정하려는 경우 대량 이메일 메시지 유형이 사용됩니다. 이러한 유형의 대량 이메일은 사업부 가입자, 특정 시장 지역 가입자 또는 특정 서비스에 보낼 수 있습니다.
신용카드 거래
가입자가 신용 카드 결제 거래를 완료하면 거래 세부 정보가 이 이메일 템플릿을 사용하여 이메일을 통해 고객에게 전송됩니다.
이벤트 메시지
이벤트 메시지 유형 이메일 템플릿은 이메일 템플릿을 정의하고 이벤트 범주에 교차 매핑하는 데 사용되며, 계정 수준에서 이벤트가 기록되면 구독자에게 이메일 알림을 트리거할 수 있는 기능을 제공합니다.
애드 혹
Adhoc 메시지 유형은 이벤트 범주 매핑과 연결되지 않은 개별 메시지를 보내는 것입니다.

이메일 발송자

Emailers 탭은 이메일 템플릿을 추가/편집하고 사업부에 대한 이메일을 예약하는 데 사용됩니다.

새 이메일 작성자 추가

새 이메일 작성자를 추가하려면 다음을 수행하십시오.

  1. 에이메일 발송자사업부 화면에서 추가() 아이콘입니다.
  2. 에서기본 세부 정보섹션에 다음과 같이 필요한 세부 정보를 입력합니다.
    1. 에제목필드에 이메일 제목을 입력합니다.
    2. 에메일 제목필드에 이메일 제목을 입력합니다.
    3. 이메일 메시지를 정의할 메시지 유형을 선택하십시오. 메시지 유형드롭 다운 목록.
    4. 에서관리자 이메일필드에 이메일을 보낼 이메일 ID를 입력합니다.
    5. 에서관리자 이름필드에 관리자 이름을 입력합니다.
    6. 선택이벤트 카테고리제공된 드롭다운 목록에서
    7. 이메일을 받을 사람의 이메일 ID를 입력하십시오. 참조그리고숨은 참조필드.
      메모: 각 이메일 ID를 쉼표로 구분하여 참조 및 숨은 참조 필드에 여러 이메일 ID를 지정할 수 있습니다.
  3. 선택하다일반 텍스트또는 HTML선택한 형식으로 이메일을 보냅니다.
  4. 에서이메일/뉴스레터 메시지섹션의 이메일 본문 텍스트 상자에 고객의 이메일에 메시지를 표시할 메시지 템플릿을 입력합니다.
  5. 딸깍 하는 소리시사텍스트 상자 영역 옆에 있는 을 눌러 일반 텍스트 또는 HTML 형식의 전자 메일 메시지를 봅니다.
  6. 클릭창조하다단추.

이메일 작성자 업데이트

  1. 다음으로 이동합니다. 이메일 발송자사업부 화면입니다.
  2. 에서이메일 템플릿 목록섹션에서 편집할 이메일 템플릿을 클릭합니다.
  3. NS 이메일 발송자 업데이트화면이 나타나면 사용 가능한 필드에서 필요한 변경을 수행하고업데이트 .
Update Emailer 화면에서 메시지 유형 및 이벤트 범주 필드를 수정할 수 없습니다.

이메일 예약

이메일을 나중에 보내도록 예약할 수 있습니다. 이 화면은 DMA 또는 서비스에 대해 지정된 시간 또는 매일 대량 이메일을 예약하는 화면입니다.

이메일 예약

  1. 에이메일 발송자화면에서이메일 템플릿탭에서 메시지 유형 이메일을 대량 이메일로 클릭합니다.
  2. 클릭이메일 예약탭. 예약된 이메일 목록이 나타납니다.
  3. 이메일을 예약하려면 추가() 아이콘을 눌러 이메일 예약 대화 상자를 엽니다.
  4. 에 이메일 ID를 입력하십시오. 메일 보내기들.
  5. 클릭실행 시간필드를 클릭하여 일정 날짜 및 시간을 선택합니다.
  6. 선택한 날짜와 시간에 매일 이메일을 예약하려면일일 일정체크박스.

예약된 이메일 삭제

  1. 클릭이메일 템플릿아래의 탭이메일 발송자왼쪽 메뉴 모음에서 탭.
  2. 제거하려는 대량 이메일(메시지 유형)을 클릭합니다.
  3. NS 이메일 발송자 업데이트화면이 나타납니다.
  4. 클릭이메일 예약탭.
  5. 삭제() 아이콘행위열에서 예약된 이메일을 삭제합니다. 예약된 이메일 목록 .

쿠폰 생성

쿠폰 생성 권한은 사용자의 BU 그룹 권한을 기반으로 합니다. 이 권한이 있는 로그인한 사용자는 쿠폰을 생성하고 편집할 수 있습니다.

새 쿠폰/바우처 추가

  1. 클릭쿠폰왼쪽 메뉴에서.
  2. 에쿠폰사업부 화면에서추가하다 () 아이콘입니다. NS 쿠폰기본 화면이 나타납니다.
  3. 딸깍 하는 소리추가하다 () 쿠폰을 생성하기 위한 아이콘입니다.
  4. 아래와 같이 신규 쿠폰/바우처 생성 화면이 나타납니다.
  5. 다음과 같이 필드에 정보를 입력합니다.
    1. 쿠폰 접두사 : 쿠폰 접두사(예: TestC, Hal, CD, Friday 등)
    2. 배치 이름 : 쿠폰 배치 이름(예: 시험쿠폰, 크리스마스 등)
    3. 소매점 : 쿠폰은 (AT&T, Vodafone, Verizon, Idea 등) 소매업체 선택에서 생성할 수 있습니다.
    4. 시작 날짜 : 사용자가 쿠폰을 사용할 수 있는 날짜
    5. 만료일 : 사용자는 쿠폰이 만료되는 날짜를 정의할 수 있습니다.
    6. 쿠폰 요청자 : 쿠폰을 요청한 요청자의 이름을 나타냅니다.
    7. 지갑 충전으로 적립됨 :
      • 지갑 충전을 위한 쿠폰을 생성하려면 토글 스위치를 클릭하여 활성화하고 금액을 입력하십시오.
      • 쿠폰을 바우처로 생성하려면 토글 스위치를 클릭하여 비활성화하십시오.
    8. 할인 유형 : 쿠폰 유형 정의 – 예: 무료, 고정, 고정, 백분율, 백분율 끄기 및 유효 기간 연장
    9. 금액/퍼센트 : 선택한 유형에 대해 금액/백분율 필드가 나타납니다.
    10. 기간 및 기간 : 쿠폰 적용 기간은 쿠폰 기간이 의미하는 각 기간 선택에 따라 일 수 또는 개월 수일 수 있습니다.
    11. 접두사를 쿠폰 코드로 만들기 : 이 체크박스를 체크하면 프리픽스가 쿠폰코드로 저장됩니다.
    12. 쿠폰 수 : 사용자는 배치에 대해 생성된 쿠폰 수를 제공할 수 있습니다. "Prefix as Coupon Code"의 경우, 쿠폰의 수는 "1"이 됩니다.
    13. 쿠폰 우선 : 비즈니스에서 생성되고 구독 및 제안 유형에 매핑되는 우선 순위에 따라 일반 쿠폰 표시를 설정하는 데 사용됩니다.
    14. 쿠폰 길이 : 생성하려는 쿠폰 코드의 길이
    15. 복수 사용 : 이 토글 스위치를 선택하면 사용자는 쿠폰에 대한 최대 사용 횟수를 정의할 수 있습니다.
    16. 최대 상환액 : 쿠폰 사용 횟수 허용
    17. 단일 사용자에 의한 다중 사용 : 확인란을 통해 고객은 단일 사용자의 최대 사용 횟수에 대해 계정에서 단일 쿠폰을 사용할 수 있습니다.
    18. 단일 사용자에 의한 다중 사용 : 확인란을 통해 고객은 단일 사용자의 최대 사용 횟수에 대해 계정에서 단일 쿠폰을 사용할 수 있습니다.
    19. 대소문자 구분 : 대문자로 쿠폰 코드를 생성하려면 확인란을 선택하십시오.
    20. 선물이다 : 체크박스를 선택하면 다른 사람에게 쿠폰을 줄 수 있습니다.
    21. 상속 가능 : 체크박스를 선택하면 쿠폰을 상속받을 수 있습니다.
    22. 모든 항목이 필수 항목입니까? : 이 확인란을 선택하면 모든 항목이 필수 항목입니다.
    23. 결제 방법 건너뛰기 : 체크박스를 선택하면 쿠폰적용 후 결제수단을 건너뛰게 됩니다.
    24. 쿠폰 설명 : 쿠폰 설명 입력
    25. 캠페인 정보 : 캠페인 정보 세부정보 입력
    26. 성공 메시지 : 사용자에게 표시될 성공 메시지를 입력하십시오.
    27. 경고 메시지 : 사용자에게 표시될 경고 메시지를 입력하십시오.
    28. 적용 제품 : 사용자는 번호를 선택할 수 있습니다. 쿠폰에 적용 가능한 제품(예: 제품에 대한 평면 30% OFF 쿠폰네온 )
    29. 적용 가능한 패키지 : 사용자는 번호를 선택할 수 있습니다. 쿠폰에 대한 응용 프로그램 패키지
    30. 제안 유형 : 사용자는 쿠폰에 대한 제안 유형을 선택할 수 있습니다.
    31. 적용 가능한 결제 모드 : 사용자는 쿠폰에 적용 가능한 결제 모드를 선택할 수 있습니다.
  6. 클릭쿠폰 생성버튼을 눌러 쿠폰을 생성합니다(쿠폰 생성 시간은 사용자가 생성한 쿠폰 수에 따라 다름).
  7. 쿠폰이 생성되면, 쿠폰이 성공적으로 생성되었습니다.메시지가 표시됩니다.

쿠폰 검색

다양한 옵션을 이용하여 쿠폰을 검색하세요. 쿠폰을 검색하려면 다음을 수행하십시오.

  1. 에쿠폰화면에서 Batch Name, Prefix, Coupon Code, Status, Expiration Date From/To로 검색할 필드 값을 입력합니다.
  2. 검색 버튼을 클릭합니다. 아래 그림과 같이 쿠폰 검색 결과가 표시됩니다.
  3. 고급 검색 활성화 토글 스위치를 클릭하여 고급 검색을 활성화합니다.
  4. 목록에 나열된 적용 가능한 제품 및 적용 가능한 지불 모드에 대한 쿠폰 배치로 검색할 수 있습니다. 쿠폰 일괄 검색은 쿠폰 일괄 검색 버튼을 클릭하세요.
  5. 검색된 쿠폰은 아래와 같이 검색 결과 목록에 표시됩니다.
